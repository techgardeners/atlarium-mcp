# Claude Code Example

Add Atlarium as a remote HTTP MCP server:

```bash
claude mcp add --transport http atlarium https://mcp.atlarium.bio/mcp
claude mcp list
claude mcp get atlarium
```

Then ask Claude Code:

```text
Use Atlarium to compare neon tetra and harlequin rasbora care requirements for a planted community aquarium.
```

This is a compatibility guide, not an official Claude endorsement.
