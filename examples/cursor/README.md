# Cursor Example

Place `mcp.json` in `.cursor/mcp.json` for a project-level configuration, or
copy its contents into `~/.cursor/mcp.json` for a global configuration.

After saving, reload Cursor and confirm the Atlarium server is connected in the
MCP settings.

Prompt:

```text
Use Atlarium to check whether Corydoras paleatus and Betta splendens are compatible.
```
