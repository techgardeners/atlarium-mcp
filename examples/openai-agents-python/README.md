# OpenAI Agents SDK Example

Use this example when you want a Python agent to discover and call Atlarium MCP
tools over Streamable HTTP.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install openai-agents
python agent.py
```

The public Atlarium MCP endpoint does not require an Atlarium API key.

## Notes

- This example documents compatibility with OpenAI Agents SDK.
- It does not claim ChatGPT public app approval.
- Compatibility and tank suggestions should be treated as advisory.
