# VS Code Example

Use `mcp.json` as `.vscode/mcp.json` for a workspace or copy the `atlarium`
server into your user MCP settings.

Prompt:

```text
Use Atlarium to suggest peaceful freshwater species for a 90 liter planted tank at 24 C with pH 6.8.
```

VS Code uses `type: "http"` for remote MCP servers.
