# Changelog

All notable public MCP changes are tracked here.

## Unreleased

- Improved GitHub repository presentation, support files and public issue
  templates.
- Added a generated GitHub social preview asset under `docs/assets/`.

## 1.0.0

- Published the public read-only Atlarium Habitat Database MCP server.
- Exposed 11 read-only tools for species, plants, products, guides,
  compatibility checks, water parameters and tank suggestions.
- Added Streamable HTTP endpoint at `https://mcp.atlarium.bio/mcp`.
- Added public server card, healthcheck, client examples and production
  validation scripts.
- Published the Official MCP Registry entry as `bio.atlarium/habitat-database`.
