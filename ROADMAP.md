# Roadmap

This roadmap is intentionally narrow: the repository remains a public read-only
MCP server for structured habitat data.

## Current Focus

- Keep the public endpoint stable and monitored.
- Complete directory claim/submission flows with accurate public metadata.
- Maintain client examples for MCP clients that support remote Streamable HTTP.
- Keep ChatGPT Apps submission notes factual without claiming public approval.

## Candidate Improvements

- Broaden public guide coverage exposed through `search_guides` and `get_guide`.
- Add more examples for agent SDKs and remote MCP clients.
- Improve directory metadata automation where directories offer public APIs.
- Expand validation fixtures for edge cases in public tool inputs.

## Out Of Scope

- Private Atlarium workspace data.
- Authenticated user operations.
- Admin APIs.
- Journal, schedule or measurement writes.
- Destructive or write-capable MCP tools.
