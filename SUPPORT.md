# Support

Thanks for using Atlarium Habitat Database MCP.

## Public Questions And Bugs

Use GitHub issues for reproducible public MCP problems:

- live endpoint or server-card issues
- tool behavior that does not match the documented public surface
- documentation or client setup examples
- directory listing metadata corrections

Please include the endpoint or tool name, steps to reproduce, expected result,
actual result and a short response preview when useful.

## Security Reports

Do not open public issues for sensitive security reports.

Report vulnerabilities to TechGardeners through the contact listed on
https://atlarium.bio.

## Scope Boundary

This repository is only for the public read-only MCP server. Atlarium account,
workspace, admin, journal, schedule, measurement and write operations are out of
scope for this server.
