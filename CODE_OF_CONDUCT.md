# Code Of Conduct

Atlarium Habitat Database MCP is a technical project for public habitat data and
AI agent interoperability. Contributors and maintainers are expected to keep
discussion respectful, precise and focused on the public MCP server.

## Expected Behavior

- Be kind and professional.
- Assume good intent while staying clear about technical evidence.
- Keep bug reports reproducible and avoid personal attacks.
- Respect the public read-only boundary of this MCP server.
- Do not post secrets, private Atlarium data or personal data.

## Unacceptable Behavior

- Harassment, threats, hate speech or discriminatory language.
- Publishing private credentials, tokens, user data or workspace data.
- Pressuring maintainers to make unsupported official-client or directory
  approval claims.
- Repeated off-topic promotion unrelated to the MCP server.

## Enforcement

Maintainers may edit, hide or remove comments and issues that violate this code
of conduct. Serious or repeated violations may lead to blocked participation in
the repository.

For sensitive concerns, contact TechGardeners through the contact listed on
https://atlarium.bio.
